# Changelog

## v0.1.0 (2026-04-04)

Initial release of `openclaw-cc-camouflage`.

### Features
- status tool: reports emulator and platform support status (machine-readable)
- doctor tool: provides diagnostic guidance for emulator prerequisites
- patch_apply: stub tool, no active patch targets
- patch_revert: stub tool, no active patch targets
- not-claude-code-emulator detection via OC_CAMOUFLAGE_EMULATOR_ROOT and OC_CAMOUFLAGE_EMULATOR_FALLBACK_PATHS
